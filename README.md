## 📌 Project Overview
A professional text-based **Hangman Game** built with Python during **CodeAlpha Internship**. Player guesses a hidden word letter by letter before the hangman is complete!

---

## ✨ Features
- 🎨 **ASCII Art** Hangman (7 stages)
- ✅ **Input Validation** (no duplicates)
- 🔄 **Play Again** option
- ❌ **Wrong Letters** tracking
- 💚 **Chances Counter** (6 attempts)
- 🏆 **Win/Lose** detection

---

## 🎯 Concepts Used
- `random` module
- `while` loops
- `if-else` conditions
- `sets` and `lists`
- Functions & clean code structure
